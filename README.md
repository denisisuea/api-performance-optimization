# Optimización del rendimiento de la API

## Descripción
Este proyecto implementa técnicas de optimización del rendimiento en una API REST utilizando Node.js y Express.

## Tecnologías
- Node.js
- Express
- PostgreSQL
- Redis
- BullMQ
- Cargador de datos

## Características
- Almacenamiento en caché de Redis
- Carga diferida (paginación)
- Cargador de datos para resolver consultas N+1
- Cola de trabajos con BullMQ

## Instalación

npm install

## Ejecutar el servidor

npm run dev

## Puntos finales

/products  
/orders
